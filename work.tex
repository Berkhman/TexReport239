\section{Постановка задачи}

На плоскости заданно множество точек. Выбрать из них такие три точки, не лежащие на одной прямой, которые составляют треугольник наименьшей площади.

\section{Алгоритм решения задачи}


\subsection{Базовые структуры данных}
Класс $\mathbf{Point}$ описывает точку, состоит из двух полей $\mathbf{x_n}$ и $\mathbf{y_n}$ типа $double$, задающих координаты точки на плоскости.


Класс $\mathbf{Set}$ описывает множество точек, состоит из двух полей: $\mathbf{k}$ типа $int$(в данной задаче $\mathbf{k}$ всегда равна 3, так как три точки образуют треугольник) и массив состоящий из $\mathbf{k}$ экземпляров класса $\mathbf{Point}$.
\subsection{Построение алгоритма}
Будем решать задачу в системе координат. С клавиатуры на вход подаётся число $\mathbf{n}$ типа $int$, количество данных точек($\mathbf{n\geqslant 3}$). Также введем переменную $\mathbf{min}$, которой будет присваиваться наименьшее значение площади. Для каждого из $\mathbf{n}$ экземпляров класса $\mathbf{Point}$ с клавиатуры считываются (или случайным образом. пользователь может выбрать вариант: вводить координаты самому с клавиотуры или предоставить компьютеру выбрать их случайным образом, для этого будет создана переменная типа $boolean$) значения переменных $\mathbf{x_n}$ и $\mathbf{y_n}$, координаты точек на плоскости. Создадим $\mathbf{C_n^3}$ $\mathbf{(C_n^3 = \frac{n!}{3! * (\, n-3 )\, !})}$ экземпляров класса $\mathbf{Set}$, состоящих из 3-х точек($\mathbf{Point}$).
С помощью метода $\mathit{square}$ получим значение площади для каждого из цэ треугольников, т.е. для каждого из $\mathbf{Set}$'ов. Опишем метод: будем считать 3 расстояния для каждого экземпляра: от точки $\mathbf{P_a}$ до точки $\mathbf{P_b}$, от точки $\mathbf{P_b}$ до точки $\mathbf{P_c}$ и от точки $\mathbf{P_c}$ до точки $\mathbf{P_a}$. Получив данные значения длины трех сторон по формуле $ L_a=\sqrt{(x_b-x_a)^2 + (y_b-y_a)^2}$-расстояние от $\mathbf{P_a}$ до $\mathbf{P_b}$(далее аналогично), посчитаем значение площади треугольника по формуле Герона: $ S = \sqrt{p(p-L_a)(p-L_b)(p-L_с)} $, где $p=(L_a+L_b+L_c)/2$. Будем каждый раз сравнивать значение площади $\mathbf{Set.square}$ со значением переменной $\mathbf{min}$ (с самого начала присвоим $\mathbf{min}$ значение площади первого $\mathbf{Set}$'a), и если новое значение меньше, то будем присваивать его переменной $\mathbf{min}$. Проверив все $\mathbf{C_n^3}$ вариантов получим конечное значение $\mathbf{min}$. 3 точки, образующие треугольник, соответствующий данному значению переменной $\mathbf{min}$, и будут искомыми.
Ответом на данную задачу являются координаты искомых точек(если есть два или более треугольников равной площади, корая являеся минимальной, тогда ответом являются несколько троек пар чисел)

Примечания: Для каждого $\mathbf{Set}$'а
площадь не должна быть нулевой, иначе данный $\mathbf{Set}$ противоречит условию(3 токчи лежат на одной пряой). То есть при создании $\mathbf{Set}$'а, если его площадь равно 0, то значение переменной $\mathbf{min}$ не меняется, а именно не становится нулевым.
