\section{Постановка задачи}

На плоскости заданно множество точек. Выбрать из них такие три точки, не лежащие на одной прямой, которые составляют треугольник наименьшей площади.

\section{Алгоритм решения задачи}


\subsection{Базовые структуры данных}
Класс $\mathbf{Dot}$ описывает точку, состоит из двух полей $\mathbf{x_n}$ и $\mathbf{y_n}$ типа $double$, задающих координаты точки на плоскости.


Класс $\mathbf{Set}$ описывает множество точек, состоит из двух полей: $\mathbf{k}$ типа $int$(задает количество точек в множестве, чаще всего равна 3) и массив состоящий из $\mathbf{k}$ экзепляров класса $\mathbf{Dot}$.
\subsection{Построение алгоритма}
Будем решать задачу в системе координат. С клавиатуры на вход подаётся число $\mathbf{n}$ типа $int$, количество данных точек($\mathbf{n\geqslant 3}$). Также введем переменную $\mathbf{min}$, которой будет присваиваться наименьшее значение площади. Для каждого из $\mathbf{n}$ экземпляров класса $\mathbf{Dot}$ случайным образом определяются значения переменных $\mathbf{x_n}$ и $\mathbf{y_n}$, координаты точек на плоскости. Создадим $\mathbf{C_n^3}$ $\mathbf{(C_n^3 = \frac{n!}{3! * (\, n-3 )\, !})}$ экземпляров класса $\mathbf{Set}$, состоящих из 3-х точек($\mathbf{Dot}$).
С помощью метода $\mathit{square}$ получим значение площади для каждого из цэ треугольников, т.е. для каждого из $\mathbf{Set}$'ов. Опишем метод: будем считать 3 расстояния для каждого экземпляра: от точки $\mathbf{D_a}$ до точки $\mathbf{D_b}$, от точки $\mathbf{D_b}$ до точки $\mathbf{D_c}$ и от точки $\mathbf{D_c}$ до точки $\mathbf{D_a}$. Получив данные значения длины трех сторон по формуле $ L=\sqrt{(x_b-x_a)^2 + (y_b-y_a)^2}$, посчитаем значение площади треугольника по формуле Герона: $ S = \sqrt{p(p-a)(p-b)(p-c)} $. Будем каждый раз сравнивать значение площади $\mathbf{Set.square}$ со значением переменной $\mathbf{min}$ (с самого начала присвоим $\mathbf{min}$ значение площади первого $\mathbf{Set}$'a), и если новое значение меньше, то будем присваивать его переменной $\mathbf{min}$. Проверив все $\mathbf{C_n^3}$ вариантов получим конечное значение $\mathbf{min}$. 3 точки, образующие треугольник, соответствующий данному значению переменной $\mathbf{min}$, и будут искомыми.

Примечания: В каждом $\mathbf{Set}$'e не должны совпадать все три значения $\mathbf{x_n}$ или $\mathbf{y_n}$, иначе данный $\mathbf{Set}$ противоречит условию, т.е. кол-во вариантов уменьшается на $\mathbf{1}$. (В таком случае кол-во перебираемых вар-тов становится равным $\mathbf{C_n^3-q}$, где $\mathbf{q}$ - кол-во таких $\mathbf{Set}$'ов).
